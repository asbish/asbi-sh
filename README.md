# Website for asbi.sh

[asbi.sh](https://www.asbi.sh/)

## Usage
Install [stack]("https://docs.haskellstack.org/") and [yarn]("https://yarnpkg.com/")

```bash
yarn install

yarn start  #development
#or
yarn build  #production
```
